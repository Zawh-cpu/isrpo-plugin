# **ITMO – как смысл жизни**

VS Code Extension

Это расширение добавляет набор небольших удобных и развлекательных команд, связанных с атмосферой ИТМО, мемами и автоматизацией простых задач.

---

## **Техническое задание**

**Проблема:**

> Жизнь слишком скучна. Иногда выгораешь от кода, а лабы в ИТМО просто убивают…

**Цель:**
Создать расширение для VS Code, которое сочетает утилитарные функции и «фановый» интерактив, облегчая работу со студенческими проектами и развлекая пользователя.

**Требования:**

1. Добавить команду для генерации базового C++ проекта (шаблонный проект).
2. Добавить команду для открытия ресурсов ПСЖ (сайт, телеграм-посты).
3. Разработать мини-игру «слот-машина» с WebView и визуальными эффектами.
4. Встроить демонстрационную команду «Hello World».
5. Обеспечить регистрацию всех команд через `package.json`.
6. Сделать проект готовым к запуску через npm + сборку TypeScript (esbuild/tsc).

**Результат:**
Расширение для VS Code с утилитарными и развлекательными функциями, полностью документированное и готовое к распространению.
